打包压缩tar、gzip、bzip2、xz、zip、unzip

gzip

gzip是为了取代compress 并提供更好的压缩比，zcat/zmore/zless用来查看压缩文件

bzip2

bzip2 比 gzip 提供更好的压缩比，bzcat/bzmore/bzless/bzgrep用来查看压缩文件

xz

虽然 bzip2 已经具有很棒的压缩比，不过显然某些自由软件开发者还不满足，因此后来还推出了 xz 这个压缩比更高的软件，但是这个压缩命令压缩速度偏慢，如果不是对压缩率有特别需求，使用bzip2就够了！这个软件的用法也跟 gzip/bzip2 几乎一模一样。

tar

tar 是一个打包工具，只是同时还实现了对 7z、gzip、xz、bzip2 等工具的支持，这些压缩工具本身只能实现对文件或目录（单独压缩目录中的文件）的压缩，没有实现对文件的打包压缩。（配合上述压缩可以对文件目录进行打包并压缩）

zip 命令为打包压缩命令，一般为了跟 windows 交互才会用的，不涉及 windows 推荐使用tar配合 gzip,bzip2,xz

zip 压缩
unzip 解压
